Technical Skills

Programming Languages

  • Manual testing is the process of manually checking software for defects without using automation tools. It helps ensure the application works correctly, is user-friendly, and meets requirements by executing test cases step by step.

  • Core Java is the fundamental part of Java used for building applications and automation frameworks. It includes OOPs concepts, exception handling, and file handling, making it essential for automation testing with tools like Selenium.

  • Automation testing uses software tools to execute test cases automatically, reducing manual effort and improving efficiency. It helps in faster, repeated testing and is commonly done using tools like Selenium, TestNG, and JUnit.

  • SQL is used in software testing to manage and validate database data. It helps testers retrieve, update, and verify records to ensure data integrity and accuracy in applications.

  • An API (Application Programming Interface) allows different software systems to communicate and exchange data. API testing ensures functionality, reliability, and security using tools like Postman and Rest Assured.

  • Tosca is a popular automation testing tool used for end-to-end testing of web, mobile, and desktop applications. It provides scriptless automation, making it easy for testers to create and execute test cases without coding.

Technical Tools

  • Eclipse is a widely used IDE for Java development and automation testing, supporting tools like Selenium and TestNG.

  • SQL*Plus is an Oracle command-line tool used to execute SQL queries, manage databases, and perform administrative tasks.

  • Selenium is an open-source automation testing tool used for testing web applications across different browsers and platforms. It supports multiple programming languages like Java, Python, and C# for writing test scripts.

  • TestNG is a testing framework for Java that helps in organizing, executing, and generating reports for automated test cases.

  • Postman is a popular API testing tool used to send requests, validate responses, and automate API testing.

  • Tosca is a scriptless automation testing tool used for end-to-end testing of web, mobile, and desktop applications.
